Programming and Data Analysis Skills

Good scientific computing skills, ideally with experience of automation through scripting languages (such as Perl and python) and of numerical analysis

Handling the data used to constrain and drive this model has provided a wealth of experience with shell scripting for data file organisation, and statistical and GIS programming for analysis (e.g Python, R, Matlab, IDL, ArcGIS and its open source equivalents).
